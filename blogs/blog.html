1. Map

In JavaScript, the map() function is used to create a new array by calling a specific function on each element present in the parent array. Here is an example of how to use the map() function in JavaScript:


const numbers = [1, 2, 3, 4, 5];
const doubledNumbers = numbers.map((number) => {
return number * 2;
});
console.log(doubledNumbers);

In this example, the map() function is used to callback function returns the new value for each element, which is then added to the new array.

2. filter()

In JavaScript, the filter() method is used to create a new array with all elements that pass the test implemented by the provided function.Here is an example of how to use the filter() function in JavaScript:


const numbers = [1, 2, 3, 4, 5];
const filteredNumbers = numbers.filter((number) => {
return number > 2;
});
console.log(filteredNumbers);

In this example, the filter() function is used to callback function returns a boolean value, which determines whether the element should be included in the new array.

3. Find()


const numbers = [1, 3, 4, 5, 7, 8, 9];
const firstEven = numbers.find((number) => number % 2 === 0);
console.log(firstEven); // Output: 4

In this example, find() method is useful when you need to find a specific element in an array that satisfies a certain condition.

4. forEach()

The forEach() method is useful when you need to perform an action on each element in an array

const numbers = [1, 2, 3, 4, 5];
numbers.forEach((number) => {
console.log(number);
});

In this example,forEach function logs the current element to the console on each iteration of the loop.

7. Eventloop

Event loop is just a medium who keeps a good communication with Call Stack and Callback Queue. It checks if the call stack is free, then lets know the callback queue. Then Callback queue passes the callback function to Call stack to be executed. When all the callback functions are executed, the call stack is out and global execution context is free.
